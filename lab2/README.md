# Лабораторная работа №2: Создание первой web-страницы и работа со статикой

## Описание проекта
В данной работе реализовано веб-приложение на базе фреймворка Django, демонстрирующее навыки работы с:
* Маршрутизацией запросов (URL-адреса).
* Функциями-представлениями (Views).
* HTML-шаблонами (Templates).
* Статическими файлами (CSS, изображения).

## Структура проекта
* **helloworld/** — ядро проекта с основными настройками (`settings.py`, `urls.py`).
* **flatpages/** — приложение для обработки статичных страниц.
  * **templates/** — HTML-шаблоны (`index.html`, `static_handler.html`).
  * **static/** — статические ресурсы (стили CSS и логотип).

## Выполненные задачи
В ходе работы были выполнены следующие требования методических указаний:
1. **Настройка маршрутов**: реализован доступ к странице по адресам `/` и `/hello/`.
2. **Шаблонизация**: создан базовый шаблон `index.html` и расширенный `static_handler.html` с использованием тега `{% load static %}`.
3. **Работа с таблицами**:
   * Таблица расширена до 5 строк и 4 столбцов.
   * Удалены границы таблицы через CSS (`border: none`).
   * Установлена ширина таблицы 100% экрана.
4. **Стилизация и верстка**:
   * Для заголовка первого уровня (`h1`) установлен шрифт с засечками (`serif`).
   * Заголовки списков переведены на уровень `h4` с размером шрифта 14px.
   * Настроен фон страницы и семейства шрифтов без засечек для основного текста.
5. **Медиаконтент**:
   * Интегрирован логотип сервиса.
   * Через CSS высота изображения зафиксирована на уровне 30px.

## Инструкция по запуску
Для запуска проекта необходимо наличие установленного Python 3 и Django.

1. Перейдите в директорию проекта:
   ```bash
   cd helloworld
   ```
2. Выполните миграции базы данных:
   ```bash
    python manage.py migrate
   ```
3. Запустите локальный сервер:
   ```bash
    python manage.py runserver
   ```
4. Откройте браузер и перейдите по адресу:
   ```bash
    http://127.0.0.1:8000/
   ```
