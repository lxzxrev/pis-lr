# Лабораторная работа №7: Основы JavaScript и манипуляции DOM

## Описание проекта
В рамках данной работы проект был дополнен клиентскими сценариями на языке JavaScript. Основное внимание уделено работе с динамическими структурами данных и интерактивному изменению интерфейса без перезагрузки страницы.

## Основные возможности
* **Работа с массивами (`helloworld.js`)**:
    * Реализована структура данных для хранения информации о студентах.
    * Создана функция форматированного вывода таблицы в консоль браузера.
    * Добавлена логика фильтрации данных по заданному критерию (группе).
* **Интерактивность интерфейса (`fold-post.js`)**:
    * Реализован механизм «сворачивания» постов для экономии места на странице.
    * Использование `addEventListener` для отслеживания кликов по элементам.
    * Оптимизированное управление состоянием через переключение CSS-класса `.folded`.
* **Оптимизация CSS**:
    * Скрытие элементов (автор, дата, текст) реализовано через каскадные стили, что снижает нагрузку на JavaScript и делает анимацию более стабильной.

## Технический стек (2026 Edition)
* **Frontend**: Vanilla JavaScript (ES6+), CSS3.
* **Backend**: Django 6.0.3, Python 3.14.
* **Инструменты**: Chrome DevTools для отладки скриптов и мониторинга загрузки статических файлов.

## Структура папок (Static)
Для корректной работы статики файлы расположены следующим образом:
```text
articles/static/
├── css/
│   └── article.css
├── js/
│   ├── helloworld.js
│   └── fold-post.js
```

## Инструкция по запуску и проверке
1. Перейдите в папку проекта:
   ```bash
   cd blog
   ```
2. Примените миграции (Django создаст системные таблицы для пользователей):
   ```bash
   python manage.py migrate
   ```
3. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

4. **Откройте главную страницу и проверьте консоль разработчика (F12 -> Console):**
    * Там должна отобразиться таблица студентов и результат фильтрации.

5. **Проверьте интерактивность:**
    * Нажмите кнопку **«свернуть»** у любого поста.
    * Убедитесь, что контент скрывается, а текст кнопки меняется на **«развернуть»**.
