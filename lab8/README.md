# Лабораторная работа №8: Интеграция jQuery и визуальные эффекты

## Описание проекта
Восьмой этап разработки посвящен внедрению популярной JavaScript-библиотеки **jQuery**. Это позволило значительно упростить манипуляции с DOM-деревом и реализовать плавные анимации, которые делают интерфейс блога более современным и отзывчивым.

## Основные возможности
* **jQuery Integration**: Библиотека подключена через локальную копию.
* **Интерактивная подсветка постов**:
    * В каждый блок статьи добавлен слой `.one-post-shadow` с абсолютным позиционированием.
    * Реализована плавная анимация прозрачности (`opacity`) при наведении курсора с помощью метода `.hover()`.
* **Анимация логотипа**:
    * Реализован эффект «пульсации» при наведении на основной логотип.
    * Использована функция `.animate()` для плавного изменения размеров (ширины) элемента.
* **Обработка событий**: Отработано использование `event.currentTarget` для точного определения элемента, инициировавшего событие во вложенных структурах.

## Структура статических файлов
```text
articles/static/js/
├── helloworld.js      # Базовая логика на чистом JS
├── fold-post.js       # Сворачивание постов
├── highlight-post.js  # Анимации на jQuery (подсветка и лого)
└── (jquery.js)        # Локальная копия
```
## Технический стек (2026 Edition)
* **Library:** jQuery 4.0.0
* **Frontend:** JavaScript (ES6+), CSS3 (Z-index, Opacity animations)
* **Backend:** Django 6.0.3, Python 3.14

## Инструкция по запуску и проверке

1. Перейдите в папку проекта:
   ```bash
   cd blog
   ```
2. Примените миграции (Django создаст системные таблицы для пользователей):
   ```bash
   python manage.py migrate
   ```
3. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

4. **Проверьте подсветку статей:**
    Наведите курсор на любую статью в списке. Она должна плавно подсветиться (появится мягкий темный фон за текстом).

5. **Проверьте анимацию логотипа:**
    Наведите курсор на логотип "Django public blog". Он должен плавно увеличиться в размере, а при уходе курсора — вернуться в исходное состояние.
