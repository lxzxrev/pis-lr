# Лабораторная работа №9: Эффект параллакса и сложная анимация

## Описание проекта
Девятый этап разработки посвящен созданию визуальной глубины интерфейса с помощью эффекта **параллакса**. Мы реализовали многослойную анимацию, где элементы страницы движутся с разной скоростью при прокрутке, имитируя перспективу и объем.

## Основные изменения и возможности
* **Дизайн (UI/UX)**:
    * Полностью обновлена цветовая гамма главной страницы (пастельные тона `#b0e0d0` и `#a0d8d0`).
    * Изменена структура шапки: теперь она включает текстовую область «Коллективный блог», блок навигации и зону декоративных иконок.
* **Технология Параллакса**:
    * Реализовано три независимых слоя декоративных элементов (`.icons-for-parallax img`).
    * **Математика движения**: Скорость каждого слоя рассчитывается по формуле `yPosition = scrolled * 0.15 * (i + 1)`, что заставляет ближние элементы двигаться быстрее дальних.
* **Интерактивный логотип**:
    * Добавлен индивидуальный эффект параллакса для основного логотипа сервиса, что придает динамику текстовой области шапки.
* **Оптимизация производительности**:
    * Использовано кэширование селекторов jQuery (`$parallaxElements`) вне обработчика события `scroll` для минимизации нагрузки на CPU при скролле.

## Технический стек (2026 Edition)
* **Backend**: Django 6.0.3, Python 3.14.
* **Frontend**: jQuery 4.0.0, CSS3 (Flexbox, Float, Absolute Positioning).
* **Скрипты**: `parallax.js` — основной контроллер смещения координат.

## Инструкция по запуску и проверке

1. Перейдите в папку проекта:
   ```bash
   cd blog
   ```
2. Примените миграции (Django создаст системные таблицы для пользователей):
   ```bash
   python manage.py migrate
   ```
3. Запустите сервер:
   ```bash
   python manage.py runserver
   ```
4. Проверьте визуальный эффект:
Откройте главную страницу http://127.0.0.1:8000/. При прокрутке (скролле) страницы вниз три иконки в правой части шапки должны начать движение «вдогонку» за пользователем с разной скоростью.

5. Проверьте логотип:
Логотип "Django public blog" также должен плавно смещаться по вертикали, создавая эффект «плавания» в пространстве шапки.
